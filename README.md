# Alphagestion

Alphagestion est une application mobile sous android
elle est conçut pour un chef d'entrepot afin qu'il puisse passer une commande vers l'entrepot principal
	Fonctionnalité
			Creation commande
			Emission de commande
			Validation de bon de sortie
			
			Redistribution du materiel sur le chantier
			
	Technologie
			REST Api PHP
			Database MySQL
